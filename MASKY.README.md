# Cinema System
This system stores movie titles with their ticket prices and allows the user to look up any movie.
It also demonstrates how to add new movies and update existing prices inside a dictionary.

## Features
- Stores movies and prices in a dictionary
- Prints the price of a specific movie (“spiderman”)
- Adds a new movie to the system
- Updates the price of an existing movie
- Lets the user search for any movie by name
- Shows the price if the movie exists
- Gives a clear message if the movie is not in stock

# How to use
Run the system and type the name of any movie.
The program will display the ticket price.
If the movie is not available, it will show a message.


## Learning Purpose
This project was created as part of backend engineering practice.
The goal was to improve skills in:
- dictionaries
- updating and adding data
- input handling
- conditional logic
